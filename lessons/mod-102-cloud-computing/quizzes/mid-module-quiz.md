## Section 1: Cloud Architecture for ML (5 questions)

### Q1. What is the primary advantage of cloud-native ML architectures?

a) Lower total cost than on-premises
b) Elasticity and scalability on demand
c) Faster training times guaranteed
d) No need for monitoring

**Answer:** B

---

### Q2. Which storage type is best for training datasets in the cloud?

a) Block storage (EBS)
b) Object storage (S3, GCS)
c) Relational database
d) Local SSD only

**Answer:** B

---

### Q3. What is the purpose of a VPC in cloud ML infrastructure?

a) To reduce costs
b) To provide isolated network environment
c) To speed up training
d) To store models

**Answer:** B

---

### Q4. Which compute option is most cost-effective for batch training jobs?

a) On-demand instances
b) Reserved instances
c) Spot/preemptible instances
d) Dedicated hosts

**Answer:** C

---

### Q5. What is the trade-off of using serverless for ML inference?

a) Higher latency vs lower operational overhead
b) Lower cost vs higher complexity
c) Better security vs worse performance
d) No trade-offs exist

**Answer:** A
